### Work

I design service architecture and data integrations for an Electronic Flight Bag platform in commercial aviation. On the backend: ASP.NET Core APIs, MongoDB data modeling for flight plans, crew rosters, and operational records. On the frontend: Blazor UIs for dispatchers and crew coordinators. It's a niche domain with real operational constraints, which keeps the engineering interesting.

### Background

Studying for an M.Sc. in Digital Business & Software Engineering, a program I chose for its focus on practical engineering over theory. I've been writing code since my teens, starting with Java and Minecraft mods.
